Privacy

Privacy Policy

PicStruct converts uploaded visual files into editable outputs such as Mermaid, CSV, Markdown, and JSON.

Files you upload

When live AI parsing is enabled, uploaded images or pasted image URLs are sent to the configured AI provider only for the purpose of generating the requested output. Demo mode runs locally in the browser and does not parse the image.

Data we store

The MVP does not include accounts or a persistent server database. The browser may store recent outputs, free usage counters, and a paid access code locally on your device. The API may keep generated parse results in temporary server memory to reduce repeated AI calls for the same image and settings.

Hosted payment providers handle checkout and payout details. Do not enter bank, card, tax, or identity information directly into PicStruct unless a production billing flow is added.

API keys

Model provider API keys must be stored on the server or deployment platform. They should never be placed in frontend JavaScript.

Contact

For privacy questions, use the contact page after the production contact address is configured.
