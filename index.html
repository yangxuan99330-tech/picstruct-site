FAQ

Practical details before you rely on it.

Does PicStruct store my images?

The current site sends the selected image or image URL to the serverless parser so the AI model can process it. Do not upload sensitive files yet. A stronger retention policy and account controls should be added before enterprise use.

Why is there a free limit?

Every real conversion calls an AI vision model. The free limit keeps testing open while protecting API costs.

How do paid access codes work?

After payment, the buyer receives a code for Starter or Pro. The code is pasted into the pricing panel and sent with future conversion requests.

Is the output always exact?

No. PicStruct produces editable structure from visible content. Users should review labels, numeric values, branches, and confidence notes before publishing.
