What PicStruct parses

PicStruct reads visible diagram structure: boxes, node labels, arrows, branch names, groups, and rough hierarchy. The result is an editable draft that can be copied into Mermaid, Markdown documentation, or JSON workflows.

Best input quality

  • Use a screenshot where text labels are readable.
  • Crop out unrelated app chrome and empty margins.
  • Prefer diagrams with visible arrows and clear grouping.
  • Review the result before using it in technical documentation.
