How-to guide

How to convert an image to Mermaid.

2 minute workflow

1. Start with a readable image

Use the clearest diagram export or screenshot you have. Labels, arrows, decision branches, and grouped sections should be visible. Crop out unrelated browser chrome or document margins when possible.

2. Upload the image to PicStruct

Open PicStruct, upload a PNG, JPG, WEBP, or SVG image, or paste an image URL. For diagram conversion, choose the diagram mode.

3. Review the generated Mermaid

Check the preview and code output. Pay special attention to arrow direction, branch labels, small text, merged nodes, and inferred relationships.

4. Copy into your documentation

After review, copy the Mermaid code into Markdown documentation, GitHub, internal docs, or a Mermaid-compatible editor.
