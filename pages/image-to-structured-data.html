Use images as structured input

Many useful images are not just pictures. They contain process steps, chart values, system relationships, or documentation cues. PicStruct helps extract that visible structure into text formats you can edit, compare, store, or paste into another tool.

Convert an image

For diagrams

Use Mermaid or JSON when the image contains nodes, arrows, and relationships.

For charts

Use CSV or JSON when the image contains axes, categories, legends, or visible values.

For documentation

Use Markdown when you need summaries, alt-text drafts, and review notes.

Common structured outputs

  • Mermaid code for diagrams and flowcharts.
  • CSV rows for visible chart data.
  • JSON objects for downstream automation.
  • Markdown summaries for documentation and review.
